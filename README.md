# gpa-planner-pro
A data-driven web application designed to help university students manage their academic performance and simulate future GPA goals. Developed using a  Product Manager (PM) mindset, this tool focuses on solving the common pain point of manual grade tracking and the lack of "forward-looking" academic planning.
Key Features:

Dynamic GPA Engine: Built an interactive dashboard using Python and Streamlit that calculates weighted GPA in real-time based on user-inputted credits and scores.

Goal Simulation Logic: Developed a "Reverse-Calculation" algorithm that allows users to set a target GPA; the system then automatically derives the required average score for remaining credits.

Interactive Analytics: Integrated Plotly to visualize grade distributions and the impact of high-credit courses on overall performance, enabling data-informed study prioritization.

User-Centric Design: Applied MVP (Minimum Viable Product) principles to create a seamless, single-page interface with robust error handling for non-numeric inputs.

Technical Stack: * Languages & Frameworks: Python, Streamlit

Data Science Libraries: Pandas (Data Processing), Plotly (Visualization)

Deployment: GitHub, Streamlit Community Cloud
